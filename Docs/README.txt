Cypress is a project dedicated to bringing back dedicated server functionality to both Garden Warfare 1 and 2! With this, you are able to host and join custom servers and extend the player count to more than 24 players in a server!



If you're reading this, hiya! Welcome to the readme!

This is a modified version of Brekko's launcher for Cypress to include small quality of life changes and adds the ability to host dedicated servers of GW1, GW2, and BFN!

Here's a basic rundown of the launcher!

Furthest left side of the program contains options such as the Game's directory path, the game you want to launch, and whether you want to launch with or without mods!

Slightly to the right of all that contains inputs you'll need to join a server! Server doesn't have a password? That's okay! Just leave the 'Enter Server Password' box blank!

To the right, you have all inputs needed to host a server! Now, read this carefully.

You will need knowledge on port forwarding, or usage of a VLAN such as Radmin, Hamachi, or anything similar.

- Device IP refers to the IPv4 address of your machine and what it's currently connected to, Which can be found typing ipconfig in Command Prompt.

- Level refers to the level the host wants to load into first. 

- Inclusion is the setup for the level, such as game mode, the time of day, and so on

There is further explanation in setting up Level and the Inclusion in GW1LevelInfo.txt and GW2LevelInfo.txt

- Don't want your server to have a password? That's fine! Just leave the 'Set Server Password' text box blank!



For info on setting up your own server, please refer to the Hosting Readme! For more info on joining servers, please refer to the Joining Readme! If you need further assistance with anything, again, feel free to hit me up @gargos69junior. :p
