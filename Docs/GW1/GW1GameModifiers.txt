Hiya! Welcome to the Game Modifiers ReadMe for Garden Warfare 1! Now, of course, since this is the first game, there aren't as much settings that drastically change the gameplay in some form, compared to GW2 and BFN. Here's some interesting ones I could find.


SyncedGame.EnableFriendlyFire - Determines if Friendly Fire should be enabled for the game.

SyncedGame.DifficultyIndex - Determines the game's difficulty.
Multiplayer - 0 
Easy - 1 
Normal - 2
Hard - 3 
CRRRRRAAAZY!! - 4
 
BFServer.InactivityTimeOut - Determines how long, in seconds, a client is inactive for before getting kicked out.
BFServer.TeamKillCountForKick - Determines how many times you kill a teammate before getting kicked
